# Hello World in Java 👋

## 📖 Overview

The "Hello World" program is the first program most beginners write when learning Java. It demonstrates the basic structure of a Java application and how to display output on the console.

---

## 💻 Source Code

```java
class Codechef {
    public static void main(String[] args) {

        // Print Hello World!!
        System.out.println("Hello World!!");
    }
}
```

---

## 🖥️ Output

```
Hello World!!
```

---

## 📝 Explanation

- `class Codechef` defines a Java class.
- `public static void main(String[] args)` is the entry point of every Java program.
- `System.out.println()` prints text to the console.
- Statements in Java end with a semicolon (`;`).

---

## 📚 Key Concepts

- Java Program Structure
- Main Method
- Console Output
- Comments (`//`)
