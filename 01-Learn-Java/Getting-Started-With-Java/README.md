# Hello World in Java 👋

## 📖 Overview

The "Hello World" program is the first Java program that introduces the basic structure of a Java application and demonstrates how to print output to the console.

---

## 💻 Source Code

```java
class Codechef {
    public static void main(String[] args) {

        // Print Hello World!!
        System.out.println("Hello World!!");
    }
}
```

---

## 🖥️ Output

```
Hello World!!
```

---

## 📝 Explanation

- `class Codechef` defines a Java class.
- `public static void main(String[] args)` is the entry point of every Java program.
- `System.out.println()` prints text to the console.
- Every Java statement ends with a semicolon (`;`).

---

## 📚 Key Concepts

- Java Program Structure
- Main Method
- Console Output
- Comments (`//`)
